
# Delivery Label Generator

A lightweight Flask application that generates and downloads delivery labels as PDF files.

---

## Setup Guide

1. **Create Project Folder**  
   Create a folder named `delivery-dashboard` and navigate into it.

2. **Set Up Virtual Environment**  
   Initialize a Python virtual environment and activate it.

3. **Install Dependencies**  
   Ensure `requirements.txt` is in the project directory, then install all required packages.

4. **Add Application Files**  
   - Add `app.py` to the root of `delivery-dashboard`.
   - Create a `templates` folder.
   - Place `login.html` and `dashboard.html` inside the `templates` folder.

---

## Running the Application

Activate your virtual environment and run the application using Python.  
Once running, open your browser and go to: [http://127.0.0.1:5000](http://127.0.0.1:5000)
